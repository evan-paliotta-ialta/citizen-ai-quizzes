# Quiz — Module 11: Documents, Images, and Multimodal Input

**Format**: Multiple choice + short answer
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
Which of the following file types can Claude NOT work with directly in Claude Desktop?

A) PDF
B) PNG image
C) Excel .xlsx file (uploaded directly)
D) Pasted table from a spreadsheet

**Correct answer**: C
*(Note: Excel files must be copy-pasted as data; Claude cannot open .xlsx files natively. Pasted tabular data works well.)*

---

**Question 2**
You upload a 90-page industry report to Claude and ask "what do you think?" What is the likely result?

A) A highly detailed analysis of every section
B) A generic summary, because Claude was not given a specific question to focus on
C) An error — Claude cannot handle documents that long
D) Claude will ask you clarifying questions before proceeding

**Correct answer**: B

---

**Question 3**
After a strategy meeting, you take a photo of the whiteboard covered in notes and diagrams. The most effective way to use this in Claude is:

A) Describe the whiteboard contents in text
B) Upload the photo and ask Claude to extract the key decisions, open questions, and action items
C) Transcribe the whiteboard manually before uploading
D) Whiteboard photos are too low-resolution for Claude to read

**Correct answer**: B

---

**Question 4**
Why should you upload only the relevant section of a large document rather than the full document?

A) Claude cannot process documents over 20 pages
B) Claude charges more for longer documents
C) Large documents consume context window space, leaving less room for nuanced responses
D) Claude always ignores pages beyond the first 10

**Correct answer**: C
