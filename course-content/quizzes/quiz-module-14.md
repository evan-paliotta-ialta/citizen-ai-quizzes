# Quiz — Module 14: Advanced Claude — MCP, Agents, and RAG

**Format**: Multiple choice + short answer
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
An MCP server is best described as:

A) A more powerful version of Claude that runs on a server
B) A connector that allows Claude to read from and write to an external tool directly
C) A backup copy of your Claude conversations stored in the cloud
D) A security layer that prevents Claude from accessing sensitive data

**Correct answer**: B

---

**Question 2**
What is the key difference between a Claude assistant and a Claude agent?

A) Agents use a more powerful model than assistants
B) An assistant generates responses; an agent takes autonomous actions in external systems
C) Agents work in real time; assistants have a knowledge cutoff
D) There is no meaningful difference — they are different names for the same thing

**Correct answer**: B

---

**Question 3**
You want Claude to accurately answer questions about your company's internal processes, past client proposals, and proprietary research — information that is not in Claude's training. The right solution is:

A) Use Opus — it has broader knowledge
B) RAG — build a system that retrieves relevant internal documents and adds them to Claude's context before answering
C) Train a custom version of Claude on your company's data
D) Paste all your internal documents into every conversation

**Correct answer**: B

---

**Question 4**
You are designing an agent that automatically sends client follow-up emails after a sales call. What is the most important design principle to apply?

A) Use the fastest model to minimize delay
B) Build in a human review checkpoint before the email is sent
C) Give the agent access to as many tools as possible for flexibility
D) Ensure the agent runs at least once per hour to stay current

**Correct answer**: B
