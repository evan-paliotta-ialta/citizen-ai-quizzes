# Quiz — Module 8: Tips, Tricks, and Power User Habits

**Format**: Multiple choice + short answer
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
Your Claude Desktop window closes unexpectedly during an important session. When you reopen it and start a new conversation, what is the best way to recover your context?

A) Retype everything from memory
B) Paste the handoff brief you saved at the end of the previous session
C) Start over — context cannot be recovered
D) Ask Claude to check its history from previous sessions

**Correct answer**: B

---

**Question 2**
You ask Claude to review a client proposal "as both a skeptical investor and a first-time reader who does not know our product." This technique is called:

A) Role assignment
B) Persona stacking
C) Context loading
D) Constraint framing

**Correct answer**: B

---

**Question 3**
You want Claude to stop beginning every response with "Certainly!" and stop using the word "leverage." The best approach is:

A) Switch to a different model
B) Add negative instructions: "Do not start with 'Certainly!' and do not use the word 'leverage'"
C) Start a new conversation each time to reset defaults
D) Edit Claude's responses manually after receiving them

**Correct answer**: B

---

**Question 4**
Before starting a complex task, you ask Claude: "What information would help you produce the best possible output?" This technique helps you:

A) Save tokens by reducing the length of your prompt
B) Surface gaps in your brief before Claude starts guessing
C) Select the correct model for the task
D) Prevent Claude from asking clarifying questions mid-task

**Correct answer**: B

---

**Question 5 — Short Answer**
Describe one prompt that has worked well for you (or that you can imagine working well for a task in your role). Write out the full prompt and explain what makes it effective.
