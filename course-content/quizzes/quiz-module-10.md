**Question 1**
What problem do Claude Desktop Projects primarily solve?

A) They allow multiple users to share a Claude account
B) They pre-load standing context so every conversation starts with Claude already knowing your role, company, and preferences
C) They increase the context window size for long conversations
D) They allow Claude to access the internet automatically

**Correct answer**: B

---

**Question 2**
Your Project instructions are best described as:

A) A template Claude fills in each session
B) A user prompt that applies to the next conversation only
C) A system prompt that runs silently before every conversation in the Project
D) A document Claude reads only when you explicitly reference it

**Correct answer**: C

---

**Question 3**
Which of the following belongs in Project instructions?

A) The specific task you want Claude to do today
B) Your company's name, your role, your communication style preferences, and standing rules
C) The full text of every document Claude might need
D) A summary of your last conversation

**Correct answer**: B

---

**Question 4**
What is the risk of making Project instructions too long?

A) Claude will ignore instructions that exceed a certain word count
B) Long instructions consume significant context window space on every conversation and may reduce performance
C) Claude cannot process instructions longer than one page
D) There is no risk — longer instructions always produce better results

**Correct answer**: B
