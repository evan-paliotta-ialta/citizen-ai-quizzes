# Quiz — Module 16: Databases and Data Storage

**Format**: Multiple choice + short answer
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
Your company's HubSpot CRM data — contacts, deals, companies — is stored in:

A) AWS S3
B) A relational database with structured tables and relationships
C) A vector database optimized for semantic search
D) GitHub

**Correct answer**: B

---

**Question 2**
What is AWS S3?

A) A relational database for structured business data
B) A cloud object storage service for storing and retrieving files — not a database
C) Amazon's version of a data warehouse
D) A vector database for AI applications

**Correct answer**: B

---

**Question 3**
A RAG system (from Module 14) uses which type of database to store and search document embeddings?

A) A relational database
B) A time-series database
C) A vector database
D) A data warehouse

**Correct answer**: C

---

**Question 4**
How does Claude typically access live data in a connected application like HubSpot?

A) Claude directly queries the HubSpot database using SQL
B) Claude accesses HubSpot through an MCP server that acts as a bridge
C) Claude reads HubSpot exports from AWS S3
D) Claude cannot access live application data under any circumstances

**Correct answer**: B

**Question 5 — Short Answer**
Describe one data source you use regularly at work. Identify: (1) what type of storage it is (database, file storage, data warehouse), and (2) what you would want Claude to do with that data if it were connected.
