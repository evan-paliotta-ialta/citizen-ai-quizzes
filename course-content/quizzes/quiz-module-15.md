# Quiz — Module 15: GitHub — The Collaboration Layer

**Format**: Multiple choice + practical
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
A colleague says "I opened a PR for the new prompt templates." This means:

A) They created a new repository
B) They submitted a formal proposal to merge their branch's changes into the main branch, pending review
C) They downloaded a copy of the prompt library
D) They published the templates publicly

**Correct answer**: B

---

**Question 2**
You want to customize a public MCP server configuration for your team's use without affecting the original. You should:

A) Edit the files directly in the original repository
B) Fork the repository to your own GitHub account, then make changes to your fork
C) Clone the repository, make changes, and push to the original
D) Download the files, edit them locally, and email them to your team

**Correct answer**: B

---

**Question 3**
Which of the following should NEVER be committed to a GitHub repository?

A) A README file explaining the project
B) A Markdown file containing prompt templates
C) An API key for your HubSpot account
D) A configuration file with non-sensitive settings

**Correct answer**: C

---

**Question 4**
Your team's production database should be stored:

A) In a GitHub repository for version control
B) In a dedicated database system or cloud object storage — not in GitHub
C) In a private GitHub repository with restricted access
D) In the same repository as the code that accesses it

**Correct answer**: B
