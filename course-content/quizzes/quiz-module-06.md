# Quiz — Module 6: Anatomy of a Good Prompt

**Format**: Multiple choice + short answer
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
Which of the following is NOT one of the five elements of a well-structured prompt?

A) Role
B) Task
C) Urgency
D) Constraints

**Correct answer**: C

---

**Question 2**
You use Claude Desktop Projects for all your work. The instructions you set in your Project are:

A) A user prompt that applies only to the next message
B) A system prompt that runs silently before every conversation in that Project
C) A template that Claude will ask you to fill in each session
D) Only relevant when you are uploading documents

**Correct answer**: B

---

**Question 3**
You tell Claude: "Do not use any statistics unless I have provided them, and do not use the phrase 'it is worth noting.'" These belong to which element?

A) Role
B) Task
C) Constraints
D) Output Format

**Correct answer**: C

---

**Question 4**
What is the main reason to specify an output format in your prompt?

A) Claude cannot produce structured output without being told to
B) It prevents Claude from producing output you will need to reformat before using
C) Output format instructions are required for the response to be accurate
D) It reduces the number of tokens used

**Correct answer**: B

---

**Question 5 — Short Answer**
Write a full five-element prompt for a real task in your role. Label each element. The task can be anything — a document, an analysis, an email, a plan.

*(This question is graded on whether all five elements are present and whether the prompt is specific to your actual work.)*
