# Quiz — Module 3: Tokens — The Currency of AI

**Format**: Multiple choice + short answer
**Passing score**: 4/5
**Attempts allowed**: Unlimited

---

**Question 1**
What is a token?

A) A single word
B) A single character
C) A variable-length chunk of text, roughly three-quarters of a word on average
D) A unit of cost charged per Claude session

**Correct answer**: C

---

**Question 2**
You are using Claude to analyze a 60-page contract but only have a question about one specific clause. What is the best approach?

A) Upload the full contract so Claude has complete context
B) Paste only the relevant clause or section to conserve context window space
C) Summarize the contract yourself first, then paste the summary
D) Use Opus instead of Sonnet to handle the full document

**Correct answer**: B

---

**Question 3**
Midway through a long Claude conversation, Claude starts ignoring something you established clearly at the beginning. What is the most likely explanation?

A) Claude is malfunctioning
B) You need to re-state your instructions more forcefully
C) The context window is getting full and earlier information is carrying less weight
D) Claude disagreed with your earlier instruction

**Correct answer**: C

---

**Question 4**
You ask Claude to count the number of times the letter "e" appears in a paragraph. Claude gives you the wrong count. Why?

A) Claude is not capable of reading carefully
B) Claude operates on tokens, not individual characters, making precise character-level tasks unreliable
C) You need to use a different prompt
D) This only happens in Haiku, not Sonnet

**Correct answer**: B

Describe one change you could make to how you currently use Claude (or plan to use it) to be more token-efficient. Be specific.
